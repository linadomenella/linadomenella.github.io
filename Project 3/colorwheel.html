THE COLOR WHEEL INTERACTIVE WEBSITE

Presented with only the information for this conference (i.e., the title, the dates, times, presentations etc.), I was tasked to create a poster design that would draw attention to the conference and convey the necessary details. This challenge was to include all the information within the design. I ended up with this layout because there was a good sense of hierarchy within it and believe that the colors help allude to that hierarchy.

FINAL DESIGN

This was one of the first web projects that I did and I am very pleased with the way it came out. This website is targeted more towards kids so that they can learn more about the color wheel. Therefore I wanted to make it fun and interactive. I picked the color wheel because I had done a project in physics in high school all about the color wheel and thought that it was very interesting. I also love playing with colors and I wanted to make a website about something that would be interesting to me.
