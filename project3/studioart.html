DESCRIPTION

This project involved taking a picture of an object and then using that to create two paintings and two digital drawings that utilized different color combinations. I chose these color combinations because of the contrast between the colors. These color combinations uniquely highlight different features of the object.

DRAWING

DESCRIPTION

I created this drawing from a picture that I had taken of my brothers while hiking in Arizona. For this drawing I used charcoal, a blending stick, and eraser and Artist Chamois. This project was difficult as there were so many small details. However, the shading and use of dark charcoal against the light shades allow these details to come together into a coherent vision.
